# Customer Retail ML Project

## Overview

This project compares different Machine Learning models using customer retail dataset.

## Models Used

* Logistic Regression
* Decision Tree
* KNN

## Technologies

* Python
* Pandas
* Matplotlib
* Scikit-learn

## Features Used

* Quantity
* UnitPrice
* Country

## Visualizations

* Scatter Plot of Quantity vs UnitPrice
* Model Accuracy Comparison Graph

## Evaluation Metrics

* Accuracy Score
* Confusion Matrix

## How to Run

Install required libraries:

```bash
pip install -r requirements.txt
```

Run the project:

```bash
python project.py
```

## Learning Outcomes

* Data preprocessing
* Data visualization
* Model training and testing
* Confusion matrix analysis
* ML model comparison
